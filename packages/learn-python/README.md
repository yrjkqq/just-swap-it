# Learn Python

Python 基础速成学习项目（Y分钟速成X 风格），使用 `uv` 管理 Python 环境。

## 前置条件

1. 安装 [uv](https://docs.astral.sh/uv/)（已安装）
2. 编辑器安装 **Python** + **Jupyter** 插件

## 快速运行

```bash
# 整个文件运行
uv run python 01_basics.py

# 或者在编辑器中点击 "Run Cell" 按钮逐块运行
```

## 目录

- `01_basics.py` — 基础语法（变量、类型、控制流、循环、函数）
- `02_data_structures.py` — 数据结构（列表、元组、字典、集合）

随时新建更多 `.py` 文件练习！
